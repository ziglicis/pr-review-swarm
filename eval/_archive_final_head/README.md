# Archived: final-head evaluation (superseded)

These results and judgments reviewed each PR at its **final pre-merge SHA**
(`pre_merge_sha`), where most reviewer-raised issues had already been fixed by
the author during review. Recall there is dominated by unmatchable ground truth
(the "matchable GT" distinction in `results.md`). Superseded by the reviewed-head
evaluation (dataset `review_sha` = earliest reviewed commit), which reviews the
pre-fix code the reviewers actually saw. Kept for provenance.
